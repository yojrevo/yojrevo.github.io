字符串的定义

  • 1.字符串以’\ 0’为结束的标志：
    “hello mentor\ 0”
    (区分’0’和’\ 0’,ASCII编码值：’0’:48 ‘\ 0’:0 )
    (存放字符串会自动早后面加’\ 0’)

  • 2.转义：引号用\转义，\用\\转义

  • 3.字符串常量
    数组和指针的区别
    char *str=”hello world”指针指向.rdata常量区，不能修改
    char str[]=”hello world”拷贝有两份，可修改

  • 4.多字节字符串和宽字节字符串

    • 多字节字符串
      每个字符编码宽度都不等：中文两个字节
      wchar_t*wstr=L”hello world您好世界”
    • 宽字节字符串：每个字符都占2/4字节（win/linux）
      ‘\ 0’也占两个字节（字符也是一样）
      wctomb()等将宽字节字符串与多字节字符串转换
      cgar ch=a;
      wchar_t wch=L’a’;
      ‘\ 0’ L’\ 0’
    • 多字节字符串的输入输出
      char buf[100]={0};
      scanf_s(“%s”,buf,100);//无空格
      gets_s(buf,100)； //可以有空格
      printf(“%s\n”,buf)；//输出
    • 宽字节字符串的输入输出
      wchar_t s[100]={0};
      scanf_s(“%ws”,s,100);//没空格
      printf(“%ws\n”,s);
      fgetsws(s,100,stdin);//含有空格输入
      fputws(s,stdout);//输出

  • 字符串和字符数组

    • 四个字符数组
      char name1[]={‘h’,’e’,’l’,’l’,’o’};字符数组
      char name2[]={‘h’,’e’,’l’,’l’,’o’,’\0’};可以当作字符串
      char name3[]={‘h’,’e’,’\0’,’l’,’l’,’o’};作为字符串只有前三个
      char name4[]=”hello”;和看不见的0

实现一个函数，将字符串进行逆置（不调用库函数）
#include<stdio.h>
#include<string.h>
void reserve_str(char*a)
{
int i;
char tmp;
int len=strlen(a);
printf(“\n”);
for(i=0; i<len-1; i++)
{
if(i<len/2)
{
tmp=a[i];
a[i]=a[len-i-1];
a[len-i-1]=tmp;
}
}
}
int main()
{
char str[]=”c c++ c# java python julia”;
reserve_str(str);
printf(“%s”,str);
return 0;
}

1.mystrlen

(1)
size_t mystrlen1(const char*str)
{
if(str==NULL)
    return 0;
int count;
while(*str!='\0')
{
    count++;
    str++;
}
return count;
}
(2)//循环迭代  
size_t mystrlen2(const char*str)
{  
if(str==NULL)  
    return 0;  
const char*p=str;  
while(*++p);  
return (size_t)(p-str);  
}  
(3)//递归，消耗空间大，容易溢出 
size_t mystrlen3(const char*str) 
{  
if(str==NULL)  
    return 0;  
if(*str=='\0')  
    return 0;  
else  
    return 1+mystrlen3(str+1);  
}  
//三目运算符的简洁写法
size_t mystrlen4(const char*str)
{
return(*str=='\0'||str==NULL)?0:1+mystrlen4(str+1);
}

2.mystrcpy

size_t mystrcpy(char*dst, const char*src)
{
//dst=src;这样不对，不是拷贝，直接更改了指针的值
char*p=dst;
while(*p++=*src++);
*p='\0';
return dst;
}

3.mystrcmp

int mystrcmp(char *s1,char *s2)  
{  
assert(s1!=NULL&&s2!=NULL);//断言，括号内不满足就抛出异常  
while(*s1&&*s2&&(*s1==*s2))  
{  
    s1++;  
    s2++;  
}  
return *s1-*s2;  
}

1.常用Linux命令

命令

命令 说明
man XX 查看XX的帮助手册
sudo 获得root权限
pwd 查看当前目录
history 查看历史命令
ls -a/-l 显示全部/显示详细
mkdir 新建目录
rm -rf 删除(f不询问)(r文件夹)
mv a b 把a命名为b

文件

符号 说明
- 普通文件
d 目录
l 链接
b 块设备文件
c 字符设备
s 套接字设备

权限
user：group：other
r，w，x 读，写，执行
4，2，1

2.hello,world

用vim编写helloworld并调试运行

vim main.c创建,i进入插入模式
编辑完成，ctrl+[回到命令模式，：wq保存并退出
编译生成目标文件
.c: gcc -o hello main.c
.cpp: g++-o hello main.c
（试了一下，好像gcc也是可以编译cpp源文件的？）
运行当前目录下文件./hello

gcc编译选项
-o outfile输出
-c 只编译，不链接（有多个cpp文件时使用？）
-llibrary 进行链接时搜索名为library的库
-Idir 把dir加入到搜索文件的路径列表中(-I/home/user/inc/)
-Ldir把dir加入到搜索库文件的路径列表中(-L/home/user/libs/-lmath)
-g生成调试版本的程序
-Wall打开有用的警告选项

3.makefile 文件编写

规则

目标：需要的条件
（TAB）命令

1目标可以是一个或多个可以是目标文件，可以是可执行文件，可以是标签（clean）
2需要的条件就是生成目标所需要的文件或目标
3命令就是生成目标所需要执行的脚本

例子

makefile文件的编写
vim makefile

用makefile编译运行，clean清除多余的.o文件

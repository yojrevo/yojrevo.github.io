关于函数传参的一点基本认知

指针

指针是一个存储地址的变量

函数传参

为什么要区分函数传参时传的时值还是地址？
如果传递变量的值，就是拷贝一份值传到函数里去，在被调函数内对形参的修改与调用者的实参无关。
如果传递变量的地址，那么可以通过解引用(*)修改变量值。

分析一段简单代码

    int func1(char*buf1,int*buf2)
    {//(略)       
    }
    int main()
    {
        char buf1[]="hello freshman"
        int num = 10;
        func(buf1,&num);
    }

在传参时，拷贝了buf1的值和&num的值，
在func1中，可以对buf1数组元素的值和num的值更改
buf1和&num的值不可更改
